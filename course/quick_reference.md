# 📋 Quick Reference — Afroyla Email & SMS Framework

> Cheat sheet dùng nhanh khi viết. Đọc modules đầy đủ khi cần chi tiết.

---

## 🔥 Core Principle

> **"Entertain first, Empower second, Sell third."**
> Wrap broccoli (product) trong hot dog (story). KHÔNG BAO GIỜ serve broccoli trần.

---

## ✍️ 21 Style Rules (Tóm tắt)

| # | Rule |
|---|------|
| 1 | ONE story per email |
| 2 | ONE throughline |
| 3 | Michelle IN the story (not narrating from outside) |
| 4 | INHABIT scenes, don't summarize |
| 5 | Don't pre-sell ("If I told you...") |
| 6 | 100% standalone — no backstory needed |
| 7 | Save unused material for future |
| 8 | Paste RAW customer words (typos, emoji = proof) |
| 9 | Every scene needs a TURN |
| 10 | Segment-specific voice |
| 11 | Characters = architecture (build arcs) |
| 12 | Show the receipt |
| 13 | SMS = friend texting |
| 14 | SMS = one thought per line |
| 15 | Scarcity through conversation |
| 16 | SMS jokes = organic situations |
| 17 | Em dash ≤ 3 per email |
| 18 | Don't borrow details between emails |
| 19 | Opening = offer + hook in 3-5 lines |
| 20 | SMS ≠ corporate announcement |
| 21 | Real voices > manufactured stories |

---

## 📧 Email Structure

```
1. SCENE SETTER — time, place, sensory detail
2. HOOK — something unexpected
3. NARRATIVE — short paragraphs, dialogue, build
4. PIVOT — story → bigger meaning
5. CTA — matches emotional altitude
6. SIGN-OFF — "Love, Michelle"
7. P.S. — Nia fact OR George joke (mandatory)
```

## 📣 Campaign Phases

```
TEASER (7-10 days)  → Layer-by-layer reveal, drive registration
WARMUP (3-5 days)   → Insider content, screenshot leaks, game plan
SALE DAY (2-3 days) → Real-time chaos, segment-specific, SMS cadence
POST-SALE (1 day)   → Receipt proof, character reunion, gratitude
```

## 🔍 Teaser Checklist (Pre-Flight)

- [ ] Sale-centric, not story-centric
- [ ] Unique angle (not same as yesterday)
- [ ] Context block rewritten (not copy-pasted)
- [ ] 100% standalone
- [ ] ONE story, ONE throughline
- [ ] Michelle IN the story
- [ ] Scene has a TURN
- [ ] Remove story → still understand sale? ✅
- [ ] Remove sale info → story still good? ✅

---

## 🚫 Anti-AI Checklist

- [ ] Em dashes ≤ 3
- [ ] No "Here's the thing"
- [ ] No brochure triplets
- [ ] No "If I told you..." 
- [ ] No "So that's the story"
- [ ] No sentimental restatement
- [ ] No softening lines
- [ ] Read aloud = Michelle or marketer?

---

## 📱 SMS Rules

- Never say "VIP" or "EXCLUSIVE"
- One joke per SMS, one character max
- Under 320 chars
- Rotate angles (never repeat)
- End: "— Michelle 💜"
- Situations > constructed comedy
- "lol" > forced emoji
