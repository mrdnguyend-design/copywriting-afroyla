### 8.3 Mechanic Retrospective

**Mechanic used:** M02 variant (Elimination Bracket) + M01 (Text-to-register)

**Did it create intended behavior?** YES — sell-out urgency drove rapid-fire purchase decisions. Live order counts in SMS reinforced FOMO. March Madness theme was novel + culturally resonant.

**Customer feedback signals:**
- ⚠️ Oversell complaints on Small Leather Bags Day 1 (mitigated by apology email + 80% discount)
- ⚠️ AFR995E keyword confusion — customers confused by number formatting (lesson: keyword + number on separate lines)
- ✅ Strong positive engagement on Day 2 record-order social proof (escalating 7→12→27 items)

**Verdict:** Re-use **with variation**
- **When next safe:** 2026-09-20 (6 months) — but consider longer cooldown (12 months) to preserve novelty as it's a strong themed mechanic
- **What to vary:** Different sport/theme anchor (e.g., Olympic-style brackets, World Series, NCAA Final Four-style madness)
- **What to keep:** Sell-out elimination mechanic, live order count SMS pattern, real cap (not announced)

### 8.6 Lessons → Propagated

**Style/process lessons (should propagate to `learnings/`):**

1. **"Reader" → "queen" voice rule** for Afroyla (prevents Lettoria contamination) — confirmed already in `style-rules.md` workflow
2. **Handbags NEVER $9.95** — only wallets/mini-wallets/makeup bags. Math accuracy rule.
3. **Segment-exclusion language must match audience** — "Your order wasn't one of them" (when ALL recipients are unaffected) vs "if you weren't affected" (when SOME might be)
4. **Emotions need breathing room** — one emotion per line, not compressed
5. **Specific counts > vague "several"** — "5 items" not "several items" when the number is verifiable
6. **AFR995E instructions need step-by-step clarity** — keyword + number on SEPARATE lines for copy-friendly UX
7. **Math accuracy:** 12 × $9.95 = $119.40 (under $120, NOT $130) — verify all multiplication before send
